⚙️ Configuración requerida

Para usar esta app con OneDrive necesitas registrar una aplicación en Azure AD y pegar tu Client ID abajo.

1. Ve a portal.azure.com
2. Azure AD → Registros de aplicaciones → Nuevo registro
3. Nombre: "InspectPro", Plataforma: Aplicación de página única (SPA)
4. URI de redirección: https://login.microsoftonline.com/common/oauth2/nativeclient
5. Permisos: Files.ReadWrite, User.Read
6. Copia el Id. de aplicación (cliente)
